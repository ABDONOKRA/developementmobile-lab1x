# Lab 1 : HelloToast - Manipuler les composants et les événements

## Description
Ce projet est une introduction au développement Android. L'application affiche un compteur qui peut être incrémenté et un bouton pour afficher un message temporaire (Toast).

## Fonctionnalités
*   **Compteur** : Un champ texte (`TextView`) affiche une valeur numérique qui augmente à chaque clic sur le bouton "Incrémenter".
*   **Toast** : Un bouton permet d'afficher un message de bienvenue ("Bonjour !") en bas de l'écran.

## Composants UI utilisés
*   `LinearLayout` : Pour l'organisation verticale des éléments.
*   `TextView` : Pour l'affichage du compteur.
*   `Button` : Pour les interactions utilisateur.

## Apprentissages clés
*   Liaison entre le layout XML et le code Java via `findViewById()`.
*   Gestion des événements de clic avec `setOnClickListener()`.
*   Utilisation de la classe `Toast` pour les notifications rapides.
*   Manipulation dynamique du texte avec `setText()`.

## Configuration requise
*   **Langage** : Java
*   **API Minimum** : 24 (Android 7.0)
